# RepoIntel | GitHub Repository Analyzer

**RepoIntel** is a web application that analyzes any public GitHub repository and provides:

- A **score** and **level** (Beginner / Intermediate / Advanced)
- A **dynamic improvement roadmap** tailored to the repository
- Information about **languages**, **commit count**, and **repository activity**

This version includes a **modern Tailwind CSS frontend** and a **FastAPI backend**.

---

## 🚀 Features

- **Dynamic Roadmap:** Suggestions based on repository structure, commits, README, tests, CI/CD, docs, stars, and languages.
- **Commit Counting:** Efficiently counts all commits using GitHub API pagination headers.
- **Languages & Activity:** Lists all languages used and total commits.
- **Score & Level:** Provides an overall quality score and developer level.
- **Tailwind Frontend:** Modern, responsive, and user-friendly UI.

---

## 🛠 Technology Stack

- **Backend:** Python, FastAPI, Requests
- **Frontend:** HTML, Tailwind CSS, Lucide icons
- **Hosting / Deployment:** Render
